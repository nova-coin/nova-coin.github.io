NOVACOIN

It is the first cryptocurrency based on hybrid Proof-of-Work (Scrypt)/Proof-of-Stake block generation methods that enable instant, near-zero cost payments and contribute to the coin's energy efficiency.

PRIVACY AND SECURITY

NovaCoin is an open-source decentralized global payment network, it supports private transactions without the need to use master nodes or mixing services for the address starts from 5.

SPEED AND EMISSION

Despite a more frequent block generation than Bitcoin, NovaCoin uses a different emission model with lower rewards. This is designed to ensure scarcity and enhance the stock-to-flow value of the coin.
